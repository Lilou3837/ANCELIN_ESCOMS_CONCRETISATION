 
    ---   README concernant le projet de concrétisation (Faraway) de Cyril Ancelin et Lilou Escoms   ---

Le dossier principal contient, en plus de ce fichier texte, les dossiers :
 - Competitions_Results
 - Instances
 - Results
 - ProgrammesPython
 - SiteFaraway

Détails :

    SiteFaraway contient les pages html et le fichier css du site. La page principale est index.html
    Le dossier liens contient tous les fichiers/images/archive qui sont disponibles sur le site.
    Le site a été mis en ligne sur starwars à l'adresse : https://leria-etud.univ-angers.fr/~lescoms

    Competitions_Results contient les 15 fichiers des résultats des 15 instances de compétition mis sur Moodle.

    Instances, Results et ProgrammesPython servent à faire fonctionner le programme.

    Pour tester notre algorithme sur une instance de votre choix, vous devez :
      1) Enregistrer votre fichier texte instance dans le répertoire Instances
      2) Vous rendre dans le fichier Python main.py et rentrer le nom de votre fichier texte dans la ligne 
         de commande en bas du fichier.
      3) Faîtes attention à bien être placé dans le repertoire ProgrammesPython pour éxecuter main.py
      4) Votre fichier résultat sera écrit dans le répertoire Results

        Cette méthode est également écrite dans main.py
